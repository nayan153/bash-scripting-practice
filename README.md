# Bash Scripting Practice

A collection of Bash scripting exercises created while learning Linux, Shell Scripting, and DevOps fundamentals.

## About

This repository documents my hands-on journey of learning Bash scripting on Ubuntu Linux. Each script focuses on a specific concept used in automation and DevOps workflows.

## Topics Covered

### Variables and User Input

* Variable declaration
* Command substitution
* User input using `read`

File:

```text
variables.sh
```

### Command Line Arguments

* Using `$1`, `$2`, `$3`
* Creating directories from arguments
* Understanding script parameters

File:

```text
arguments.sh
```

### Conditional Statements

* if
* if-else
* Nested if
* Numeric comparison
* String comparison

File:

```text
conditions.sh
```

### Loops

* for loop
* while loop
* Directory creation automation
* Even number generation

File:

```text
loops.sh
```

### Functions and Recursion

* Function creation
* Recursive factorial program
* Parameter passing

File:

```text
functions.sh
```
